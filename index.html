About

NextLaunch is a WebApplication to see and subscribe to upcoming Rocket Launches, You can add any of these events to your Google Calender or Turn on the notifications to get notified when a rocket is about to get launched,
Here you can see upcoming rocket launches by ISRO, SPACEX, ARIANESPACE, etc.. and add them to your callender This project is using API from launchlibrary.net. You can contribute to this project by sending Pull Request to github.com/saurabhdaware/next-rocket-launch
